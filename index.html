Listado de los comando de git y sus descripciones

Primeros Comandos en Git

  1. git init: Activa git en nuestro proyecto creando un espacio en RAM llamado "STAGING" y crea una carpeta .git
  2. git add: este comando deja en staging o en estado de espera los cambios del elemento mencionado o de todos los elementos de nuestro directorio en memoria ram para luegos hacerles un commity así guardar los cambios en una nueva version del proyecto.
  3. git commit: envía los archivos con sus cambios al repositorio
  4. git status te dice el estado de tu repositorio, te advierte de guardar cambios de tus nuevos archivos y también te avisa si se realizaron cambios en alguno de ellos para que los guardes y no pierdas tus procesos. :)
  5. git log: muestra el historial de versiones o commits que se realizaron en el archivo.
  6. git show: muestra que cambios se realizaron en cada una de las versiones.
  7. git diff: compara entre las versiones del archivo.
